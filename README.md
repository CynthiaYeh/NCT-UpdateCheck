# NCT-UpdateCheck
 Python and R scripts for web scraping information from Clinicaltrials.gov

# Prerequisites
For NCT_check.ipynb:
    Python 3
    Jupyter notebook

For NCT_check.R:
    R version 4.0.4
    R studio 2022.07.1

Files needed:
    Source database.csv (Before using the script, please use augment.source 02162023_v5.R to augment source database)
